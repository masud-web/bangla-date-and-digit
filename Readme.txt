== Description ==

**Bangla Date and Digit** is a WordPress plugin that converts English days, months, and digits into the Bangla language. This plugin is ideal for Bengali-speaking users who prefer to view dates and numbers in their native script.

## Core Features

- Automatic conversion of English days of the week into Bangla.
- Automatic conversion of English months into Bangla.
- Automatic conversion of English digits into Bangla.
- Supports both full and shortened forms of days and months.
- Seamless integration with WordPress date functions.
- lightweight and does not require any special configuration after activation.

== Installation ==

1. Upload the `bangla-date-and-digit` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The plugin will automatically convert dates and digits to Bangla on your WordPress site.

== Frequently Asked Questions ==

= Does this plugin support both full and shortened forms of days and months? =
Yes, it supports both full and shortened forms of days and months.

= Which WordPress date functions does this plugin integrate with? =
This plugin integrates with various WordPress date functions such as `the_date`, `get_the_date`, `date_i18n`, `get_comment_date`, `widget_text`, `get_calendar`, `get_the_archive_title`, `get_feed_build_date`, `get_the_modified_date`, `the_modified_date`, `the_weekday_date`, `wp_date`, `the_weekday`, and `number_format_i18n`.

= Does this plugin affect the backend of my WordPress site? =
No, this plugin primarily affects the frontend display of dates and digits.

= Will this plugin slow down my site? =
This plugin is designed to be lightweight and should not significantly impact your site's performance.

= Is this plugin compatible with the latest version of WordPress? =
Yes, this plugin is tested up to the latest version of WordPress (6.5).
